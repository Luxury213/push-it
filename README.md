# Push It 

Una extensión visual para Git en VS Code. Haz commits, push y pull sin usar la terminal.

##  Características

-  **Commits visuales**: Escribe y confirma commits desde una interfaz limpia
-  **Push/Pull con un clic**: Sincroniza tu código fácilmente
-  **Información clara**: Ve tu rama actual y cambios pendientes
-  **Codigo abierto**: Puedes revisar como se hizo y hacer cambios!

##  Cómo usar

1. Abre el panel de Push It (icono en la barra lateral)
2. Escribe tu mensaje de commit
3. Haz clic en "Confirmar commit"
4. Usa Push/Pull para sincronizar

##  Instalación

Busca "Push It" en el Marketplace de VS Code o instalalo con : 
ext install push-it
